Git

git 是什么

git 是目前世界上最先进的分布式版本控制工具（没有之一）。

什么是版本控制工具

说白了就是帮助我们管理不同版本的文件的一个应用程序。

git诞生

2002年之前，linus自己手动维护Linux的版本代码。

2002-2005，使用免费的bitkeeper。

2005年后，linus开发git。

git 的作用

  1. 协同开发
  2. 版本记录
  3. 冲突解决
  4. 历史追查
  5. 代码备份
  6. 版本还原
  7. 权限管理
  8. 分支管理
  9. 代码审查
  10. 。。。

工作区、暂存区、本地库

概念

工作区(Working Directory) : 电脑的本地磁盘目录。

暂存区(stage) : 一般存储在git目录下的index文件，所以我们把暂存区也叫做 索引。

本地库(Repostory) : 工作区中有一个隐藏目录 .git，它就是git的本地版本。

冲突一般是指同一位置的文件，在两种版本的仓库合并时，版本管理软件无法判断到底应该保留哪一个版本，因此会提示该文件冲突。

什么是github
